A predictive analytics tool designed for the **subscription economy** to identify "At-Risk" customers before they churn.

### Overview
This project builds an automated pipeline that scores customers based on **RFM (Recency, Frequency, Monetary)** metrics and uses Machine Learning to predict churn probability, helping businesses protect **Customer Lifetime Value (CLV)**.

### Key Features
* **Automated Pipeline:** Transforms raw transaction logs into health scores.
* **Predictive Modeling:** Uses XGBoost to flag high-risk accounts with feature importance.
* **Live Dashboard:** Interactive UI to filter by customer segment and visualize revenue at risk.
